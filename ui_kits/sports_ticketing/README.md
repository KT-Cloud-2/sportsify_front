# Sports Ticketing UI Kit

**Product:** 스포츠 티켓팅 + 팀 채팅  
**Stack:** React (Babel inline), Pretendard font, Lucide icons via CDN  
**Design width:** 1280px desktop + 375px mobile view

## Screens
1. **Home** — 경기 목록 / 메인 페이지
2. **Match Detail** — 경기 상세 + 좌석 선택
3. **Checkout** — 결제 화면
4. **Team Chat** — 팀 채팅 (Slack-like)
5. **Admin Dashboard** — 판매 현황 대시보드

## Components
- `<NavBar>` — top navigation with sport filters
- `<MatchCard>` — game card with team, date, seats, price
- `<SeatMap>` — simplified stadium seating UI
- `<ChatPanel>` — real-time team chat panel
- `<StatCard>` — admin metric cards
- `<Badge>` — sport / status badges
- `<Button>` — all variants

## Usage
Open `index.html` — interactive prototype, click through all screens.
